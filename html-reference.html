HTML Reference Page

HTML Element

Description

Use this element at the start of your html page. Everything iside of the html elements are defined as HTML code to the browser

Syntax

<html> at the start, then </html> at the end of your code

Example

An example of when you use an HTML element is on every single html page you make.

Display

This element is not shown on the browser

Head Element

Description

Head is a part of the minimum code an HTML page should have. It goes before the body of the website. It contains metadata (data about other data)

Syntax

<head> at the top of the page and </head> at the bottom of the page

Example

The head element defines a title in the browser tab, provides a title for the page when it is added to favorites, and displays a title for the page in search engine results

Display

This element is not shown

Title Element

Description

Title is what the HTML page is called. It goes inside the head elements.

Syntax

<title>Title goes here</title>

Example

What is in the title element is what shows up inside the tab. For example, if you go to google.com, then the Google will be inside the tab.

Display

This element is shown in the tab, not on the webpage.

Body Element

Description

The body element defines what is displayed in the middle of the page.

Syntax

<body> and </body>

Example

Use a body for displaying information to the user

Display

The body element is not shown, but the code that is inside the body elements are shown to the user.
